# CLAUDE.md - Project Context for Coding Agents

## Project: lingo_pkmn
Web-based RPG for language learning targeting beginners in Spanish and French.

## Stack
- **Frontend:** HTML/CSS/JS + React
- **Backend:** Python
- **Databases:** Elasticsearch (vocab/content search) + SQL (user progress, game state, accounts)

## Game Mechanics
- NPC dialogue in the target language
- Dungeon crawler with language challenges as puzzles

## Repo Structure
- `src/` — frontend source
- `docs/` — documentation
- `.agent/` — agent notes & decisions

## Agent Notes
- Managed by Lingo 🐁
- Decisions logged in `.agent/decisions.md`
